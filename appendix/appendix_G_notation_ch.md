## G.2 噪声与传感器参数

### 噪声模型统一符号（泊松-高斯混合模型）

全书统一采用 Foi 等人（2008，TPAMI）的标准定义：

$$\sigma^2(x) = \alpha \cdot x + \sigma_r^2$$

其中 $x$ 为像素强度（线性 RAW 域，DN 值，BLC 之后）。

| 符号 | 定义 | 单位 | 首次定义章节 |
|------|------|------|------------|
| `α` | Poisson 散粒噪声系数（PTC 斜率） | DN²/DN | 第一卷第4章 §2.1 |
| `σ_r` | 读出噪声标准差（Gaussian 分量） | DN | 第一卷第4章 §2.1 |
| `σ_r²` | 读出噪声方差（PTC y 轴截距） | DN² | 第一卷第4章 §2.1 |
| `g` | 系统增益（模拟→数字转换比）；**不与 `α` 混用** | ADU/e⁻ | 第一卷第3章 §1.2 |
| `β` | 若使用：等于 `σ_r`（标准差形式，非方差）；须明确标注以避免与方差 `σ_r²` 混淆 | DN | — |

> **`α` 与 `g` 的关系**：`α ≈ g · K`，其中 `K` 为传感器量子效率与满阱容量比（或等效地 `α = 1/K_gain`，`K_gain` = e⁻/DN 系统增益）。两者物理意义不同，`g`（ADU/e⁻）描述电荷-数字转换比率，`α`（DN²/DN）描述散粒噪声在 DN 域的传播系数，不可互换。

### 通用噪声与传感器参数

| 符号 | 含义 | 上下文/备注 |
|--------|---------|----------------|
| σ²(x) | 像素强度 x 处的噪声方差 | 全书：σ²(x) = α·x + σ_r²（信号相关） |
| σ | 噪声标准差 | σ = √σ² |
| α | Poisson 散粒噪声系数（散粒噪声斜率） | 第一卷第04章：PTC斜率；单位：DN²/DN |
| σ_r | 读出噪声标准差（Gaussian 分量） | 第一卷第04章；高斯分布，信号无关；单位：DN |
| σ_r² | 读出噪声方差 | 第一卷第04章：PTC y轴截距；单位：DN² |
| β | 若使用，等于 σ_r（标准差形式）；须标注以避免与 σ_r² 混淆 | 第二卷第03章 §1.1b |
| σ_shot | 散粒噪声标准差 | σ_shot = √(α·x) |
| λ | 泊松参数（平均光子数） | 第一卷第04章：λ = QE × Φ × t_exp |
| QE | 量子效率 | 转化为电子的光子比例（0–1） |
| FWC | 满阱容量 | 饱和前每像素最大电子数；单位：e⁻ |
| DN | 数字数 | ADC计数的原始像素值 |
| ADU | 模拟-数字单元 | 等价于DN |
| g | 系统增益（ADU/e⁻）；**注意：不与散粒噪声系数 α 混用** | 第一卷第03章；单位：ADU/e⁻（或等价 DN/e⁻） |
| K | 逆系统增益（e⁻/DN） | e⁻/DN；电子数与ADC计数的换算；K = 1/g |
| DR | 动态范围 | DR = 20·log₁₀(FWC / σ_r) [dB] |
| SNR | 信噪比 | SNR = x / σ(x)（线性）；或 20·log₁₀(x/σ) [dB] |

## G.6 图像质量指标

| 符号 | 含义 | 公式/备注 |
|--------|---------|----------------|
| PSNR | 峰值信噪比（Peak Signal-to-Noise Ratio） | PSNR = 10·log₁₀(MAX²/MSE) [dB] |
| MSE | 均方误差（Mean Squared Error） | MSE = (1/N)·Σ(I_hat - I_ref)² |
| RMSE | 均方根误差（Root Mean Squared Error） | RMSE = √MSE |
| SSIM | 结构相似性指数（Structural Similarity Index） | SSIM ∈ [0,1]；越高越好；1 = 完全相同（理论范围 [-1,1]，图像质量评估实践中几乎恒为正；工程阈值：≥0.95 高质量，≥0.85 可接受）|
| MS-SSIM | 多尺度SSIM（Multi-Scale SSIM） | 在多个图像尺度下计算 |
| LPIPS | 学习感知图像块相似度（Learned Perceptual Image Patch Similarity） | 越低越好；深度特征距离 |
| MOS | 平均意见分（Mean Opinion Score） | 主观质量评分，通常1–5分制 |
| DMOS | 差分MOS（Differential MOS） | 相对于参考的畸变图像MOS |
| SRCC | Spearman秩相关系数（Spearman Rank Correlation Coefficient） | 基于秩的相关性 ∈ [−1,1]；衡量单调性 |
| PLCC | Pearson线性相关系数（Pearson Linear Correlation Coefficient） | 线性相关性 ∈ [−1,1] |
| RMSE_p | 预测分与主观分之间的RMSE | IQA模型评估指标 |
| BRISQUE | 盲/无参考图像空间质量评估器（Blind/Referenceless Image Spatial Quality Evaluator） | NR-IQA；越低越好 |
| NIQE | 自然图像质量评估器（Natural Image Quality Evaluator） | NR-IQA；越低越好 |
| NRQM | 无参考质量度量（No-Reference Quality Metric） | NR-IQA |
| FID | 弗雷歇初始距离（Fréchet Inception Distance） | GAN生成质量；越低越好 |

## G.9 歧义消除说明

| 符号 | 上下文1 | 上下文2 |
|--------|-----------|-----------|
| σ | 噪声标准差（第一卷第04章） | 神经网络中的激活函数（第三卷） |
| λ | 泊松噪声参数/平均光子数（第一卷第04章） | 光的波长（第一卷第02章）/ 正则化参数（优化） |

| f | 焦距（第一卷第02章） | MTF中的空间频率（第二卷第04章）/ 函数名 |

| W | 图像宽度 | 神经网络中的权重矩阵 |

| `α` | Poisson 散粒噪声系数（噪声模型 G.2，第一卷第04章） | 光场重聚焦参数（G.8.1，附录I §I.2.2）；时域滤波混合系数（第二卷第12章 §1.2）——均须在各上下文首次出现时明确说明含义 |
| `g` | 系统增益 ADU/e⁻（传感器物理，第一卷第03章）；**不得用作散粒噪声系数** | 模拟增益 $g_a$ 或数字增益 $g_d$ 的增益因子（第一卷第04章 §1.5）——带下标时为增益因子，不带下标时为系统增益 ADU/e⁻ |

## 习题

**练习1（理解）** 本手册中 $\sigma$ 出现在多个不同上下文中（噪声标准差、空间频率滤波器半径、高斯核参数）。参考本附录中的符号消歧规则，写出在第二卷第三章（降噪）和第三卷第四章（噪声模型）中 $\sigma$ 的具体含义各是什么。

**练习2（自测）** 查阅本附录，说明 $\text{PSNR}$ 和 $\text{SSIM}$ 的标准范围，并解释为什么对于自然图像，SSIM 实践中几乎始终为正值（接近 [0,1]），而理论定义允许 $[-1,1]$。
