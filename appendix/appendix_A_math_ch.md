## A.1 线性代数

### A.1.1 向量与矩阵

**定义。** 向量 **v** ∈ ℝⁿ 是 $n$ 个实数的有序组。矩阵 **A** ∈ ℝ^{m×n} 将向量从 ℝⁿ 映射到 ℝᵐ。

常用运算及其ISP对应关系：

| 运算 | 符号 | ISP用途 |
|-----------|----------|---------|
| 矩阵-向量乘积 | **y** = **A****x** | CCM：将3×3颜色矩阵应用于RGB三元组 |
| 转置 | **A**ᵀ | 最小二乘法正规方程 |
| 矩阵逆 | **A**⁻¹ | 求解标定线性系统 |
| Frobenius范数 | ‖**A**‖_F = √(Σᵢⱼ Aᵢⱼ²) | CCM拟合残差 |

**ISP应用 — CCM（第二卷第06章）：** 颜色校正矩阵将传感器RGB映射到参考XYZ：

```
[X]   [m11  m12  m13] [R_sensor]
[Y] = [m21  m22  m23] [G_sensor]
[Z]   [m31  m32  m33] [B_sensor]
```

### A.1.2 特征分解

**定义。** 对于对称矩阵 **A** ∈ ℝ^{n×n}，特征分解得到：

```
A = Q Λ Qᵀ
```

其中 **Q** 是正交矩阵（以特征向量为列），**Λ** = diag(λ₁, …, λₙ) 包含特征值。

当 **A** 为半正定（PSD）时，对所有 $i$ 有 λᵢ ≥ 0。

**ISP应用 — 噪声建模的主成分分析（第一卷第04章）：** 通过协方差矩阵的特征分解进行主成分分析（PCA），识别多通道传感器数据中的主要噪声方向。

### A.1.3 奇异值分解（SVD）

**定义。** 任何矩阵 **A** ∈ ℝ^{m×n} 可以分解为：

```
A = U Σ Vᵀ
```

其中 **U** ∈ ℝ^{m×m} 和 **V** ∈ ℝ^{n×n} 是正交矩阵，**Σ** ∈ ℝ^{m×n} 是与 **A** 同形的矩形对角矩阵，其对角元素 σ₁ ≥ σ₂ ≥ … ≥ σᵣ ≥ 0 为奇异值，r = rank(**A**) ≤ min(m,n)。

**低秩近似（Eckart-Young定理）：** 最佳秩-k近似为：

```
A_k = Σᵢ₌₁ᵏ σᵢ uᵢ vᵢᵀ
```

**ISP应用：**
- **LSC拟合（第二卷第08章）：** SVD求解镜头阴影曲面拟合中的病态最小二乘问题。
- **CCM拟合（第二卷第06章）：** 当测量矩阵秩亏时，基于SVD的伪逆给出最小范数解。
- **低秩去噪：** 通过截断小奇异值进行噪声抑制。

### A.1.4 范数与距离度量

| 范数 | 公式 | ISP用途 |
|------|---------|---------|
| L1 | ‖**x**‖₁ = Σᵢ |xᵢ| | 稀疏噪声惩罚 |
| L2（欧氏） | ‖**x**‖₂ = √(Σᵢ xᵢ²) | RMSE、CCM残差 |
| L∞ | max|xᵢ| | 最坏情况误差界 |
| Frobenius | ‖**A**‖_F | 矩阵拟合残差 |

**ΔE色差（第一卷第05章、第二卷第06章）：**

```
ΔE₀₀ = √[(ΔL*/k_L S_L)² + (ΔC*/k_C S_C)² + (ΔH*/k_H S_H)² + R_T(ΔC*/k_C S_C)(ΔH*/k_H S_H)]
```

## A.2 概率与统计

### A.2.1 概率分布

#### 泊松分布

**定义。** 对独立事件计数建模的离散分布：

```
P(X = k) = (λᵏ e⁻λ) / k!,    k = 0, 1, 2, ...
```

**性质：** E[X] = λ，Var[X] = λ（均值等于方差）。

**ISP应用 — 光子散粒噪声（第一卷第04章）：** 像素中收集到的光电子数量服从泊松统计。在高光子数情况下，泊松分布趋向高斯分布（中心极限定理）。

```
σ²_shot = λ = I  (信号相关噪声)
```

#### 高斯（正态）分布

**定义。**

```
f(x; μ, σ²) = (1 / √(2πσ²)) exp(-(x-μ)² / (2σ²))
```

**性质：** E[X] = μ，Var[X] = σ²。

**ISP应用 — 读出噪声（第一卷第04章）：** 热读出噪声可以用零均值高斯分布良好建模：

```
σ²_total = σ²_shot + σ²_read = α·I + σ²_read
```

这是用于去噪标定的泊松-高斯噪声模型。

#### 异方差高斯（信号相关方差）

合并后的噪声方差是信号相关的：

```
σ²(I) = α·I + β
```

其中 α 是光子转移系数，β 是读出噪声底。参见第一卷第04章§2（标定）的光子转移曲线（PTC）测量方法。

### A.2.2 贝叶斯估计

**贝叶斯定理：**

```
P(θ | x) = P(x | θ) · P(θ) / P(x)
```

- **似然** P(x | θ)：给定参数观测数据的概率。
- **先验** P(θ)：关于参数分布的先验信念。
- **后验** P(θ | x)：观测数据后的更新信念。

**MAP（最大后验）估计：**

```
θ_MAP = argmax_θ log P(x | θ) + log P(θ)
```

**ISP应用 — 去噪（第二卷第03章）：** 带高斯噪声似然和全变分（TV）先验的MAP去噪：

```
x̂ = argmin_x (1/2σ²)‖y - x‖² + λ·TV(x)
```

**ISP应用 — AWB（第二卷第05章）：** 贝叶斯颜色恒常性在给定观测图像统计量和自然光源先验的情况下估计光源概率。

### A.2.3 期望与方差

| 性质 | 公式 |
|----------|---------|
| 期望的线性性 | E[aX + bY] = aE[X] + bE[Y] |
| 方差 | Var[X] = E[X²] - (E[X])² |
| 协方差 | Cov[X,Y] = E[(X-μX)(Y-μY)] |
| 协方差矩阵 | **Σ** = E[(**x**-**μ**)(**x**-**μ**)ᵀ] |

## A.3 信号处理

### A.3.1 卷积

**二维离散卷积：**

```
(f * g)[m,n] = Σₖ Σₗ f[k,l] · g[m-k, n-l]
```

二维卷积满足以下性质：
- 交换律：f * g = g * f
- 结合律：f * (g * h) = (f * g) * h
- 对加法的分配律：f * (g + h) = f*g + f*h

> **注意（深度学习上下文）：** 深度学习框架（PyTorch、TensorFlow等）中所称的"卷积层"在实现上为**互相关（cross-correlation）**，即核不翻转：(f ⋆ g)[m,n] = Σₖ Σₗ f[k,l] · g[m+k, n+l]。由于网络权重通过反向传播学习，翻转与否不影响最终结果，但与严格数学卷积定义存在区别。本手册第三卷涉及DL ISP时"卷积"均指互相关。

**ISP应用 — 锐化（第二卷第04章）：** 非锐化掩蔽使用：

```
I_sharp = I + α · (I - I * G_σ)
```

其中 G_σ 是高斯模糊核。

**ISP应用 — 去噪（第二卷第03章）：** 线性空间滤波器（均值、高斯）是卷积。双边滤波器将其扩展为保边滤波：

```
I_bilateral[p] = (1/W_p) Σ_q G_s(‖p-q‖) · G_r(|I[p]-I[q]|) · I[q]
```

### A.3.2 傅里叶变换

**连续傅里叶变换：**

```
F(ω) = ∫ f(t) e^{-jωt} dt
f(t) = (1/2π) ∫ F(ω) e^{jωt} dω
```

**离散傅里叶变换（DFT）：**

```
X[k] = Σₙ₌₀^{N-1} x[n] e^{-j2πkn/N}
```

**卷积定理：** 空间域中的卷积 = 频域中的乘积：

```
F{f * g} = F{f} · F{g}
```

**图像的二维DFT：**

```
F[u,v] = Σₘ Σₙ I[m,n] e^{-j2π(um/M + vn/N)}
```

**ISP应用：**
- **锐化（第二卷第04章）：** 高频提升的频域视角。
- **LSC分析（第二卷第08章）：** 检测低频渐晕分量。
- **去马赛克混叠分析（第二卷第02章）：** CFA采样在奈奎斯特频率处产生混叠，在频域中可见。

### A.3.3 调制传递函数（MTF）

**定义。** MTF是归一化光学传递函数（OTF）的幅值：

```
OTF(f) = FT{PSF}
MTF(f) = |OTF(f)|
```

其中PSF（点扩散函数）是光学/传感器系统的空间脉冲响应。

**倾斜边缘MTF测量（ISO 12233）：** 边缘倾斜约5°以提供亚像素采样。边缘扩散函数（ESF）微分得到线扩散函数（LSF），再进行傅里叶变换得到MTF。

```
ESF(x) → diff → LSF(x) → FT → MTF(f)
```

**MTF50：** MTF降至峰值50%处的空间频率。关键清晰度指标。参见第二卷第04章§5（评测）。

**ISP应用：**
- **锐化（第二卷第04章）：** 非锐化掩蔽提升中高频的MTF。
- **去马赛克（第二卷第02章）：** 去马赛克算法质量部分由MTF50保留程度衡量。
- **镜头标定（第一卷第02章）：** 在多个视场位置测量MTF50用于镜头特性描述。

### A.3.4 采样与奈奎斯特定理

**奈奎斯特-香农采样定理：** 最大频率为 f_max 的带限信号可以从采样率 f_s ≥ 2·f_max 的样本中完美重建。

**ISP应用 — CFA去马赛克混叠（第二卷第02章）：** Bayer CFA以G通道一半的空间分辨率对R和B通道进行采样。当场景空间频率超过欠采样通道的奈奎斯特极限时发生混叠。光学抗混叠滤波器（OLPF）在采样前对信号进行带限处理。

## A.4 优化

### A.4.1 最小二乘

**线性最小二乘：** 给定超定系统 **A****x** = **b**（$m > n$ 个方程）：

```
x̂ = argmin_x ‖Ax - b‖²
```

**正规方程解（要求 A 满列秩，使 AᵀA 可逆）：**

```
x̂ = (AᵀA)⁻¹Aᵀb
```

**通过SVD求伪逆：** 当 **A** = **U**Σ**V**ᵀ 时：

```
x̂ = A⁺b = V Σ⁺ Uᵀ b
```

其中 Σ⁺ 仅对非零奇异值取倒数。即使对病态系统也数值稳定。

**加权最小二乘：**

```
x̂ = argmin_x (Ax - b)ᵀ W (Ax - b)
   = (AᵀWA)⁻¹AᵀWb
```

**ISP应用 — CCM拟合（第二卷第06章）：** 对N个色块测量拟合3×3矩阵：

```
M̂ = argmin_M Σᵢ wᵢ · ΔE²(M · x_i^{sensor}, x_i^{reference})
```

权重 wᵢ 可以编码色块的可靠性或感知重要性。

**ISP应用 — LSC拟合（第二卷第08章）：** 对平场图像的渐晕测量拟合平滑多项式曲面：

```
G(x,y) = Σₖ aₖ φₖ(x,y)
â = argmin_a ‖Φa - g‖²
```

其中 φₖ 是多项式基函数，g 是逐像素增益测量值。

### A.4.2 正则化

**Tikhonov（L2）正则化：**

```
x̂ = argmin_x ‖Ax - b‖² + λ‖x‖²
   = (AᵀA + λI)⁻¹Aᵀb
```

**L1正则化（LASSO）：**

```
x̂ = argmin_x ‖Ax - b‖² + λ‖x‖₁
```

促进稀疏性；通过迭代软阈值（ISTA/FISTA）求解。

**全变分（TV）正则化：**

```
TV(x) = Σᵢⱼ √((∂x/∂i)² + (∂x/∂j)²)
```

促进分段平滑解；在抑制噪声的同时保留边缘。

**ISP应用：** TV正则化用于去噪（第二卷第03章）、去马赛克（第二卷第02章）和HDR色调映射（第二卷第07章）。

### A.4.3 梯度下降与深度学习训练

**梯度下降更新规则：**

```
θ_{t+1} = θ_t - η · ∇_θ L(θ_t)
```

**随机梯度下降（SGD）：** 使用小批次梯度估计：

```
θ_{t+1} = θ_t - η · ∇_θ L(θ_t; x_batch)
```

**Adam优化器：**

```
m_t = β₁ m_{t-1} + (1-β₁) g_t            (一阶矩，梯度的指数移动平均)
v_t = β₂ v_{t-1} + (1-β₂) (g_t ⊙ g_t)   (二阶矩，梯度逐元素平方的指数移动平均)
m̂_t = m_t/(1-β₁ᵗ),  v̂_t = v_t/(1-β₂ᵗ)  (偏差校正)
θ_{t+1} = θ_t - η · m̂_t / (√v̂_t + ε)
```

> 其中 `⊙` 表示逐元素（Hadamard）乘积；`√v̂_t` 同样为逐元素开方。

默认值：β₁=0.9，β₂=0.999，ε=10⁻⁸。

**深度学习ISP的损失函数（第三卷第01–02章）：**

| 损失 | 公式 | 应用 |
|------|---------|-------------|
| L1损失 | Σ|ŷ-y| | 对离群值鲁棒 |
| L2 / MSE | Σ(ŷ-y)² | 最小化PSNR |
| SSIM损失 | 1 - SSIM(ŷ,y) | 结构相似性 |
| 感知损失 | ‖φ(ŷ)-φ(y)‖² | VGG特征匹配 |
| GAN对抗损失 | -log D(ŷ) | 感知清晰度 |

**ISP应用 — DL ISP训练（第三卷）：** 端到端去噪和复原网络使用组合损失训练，通常为L1 + 感知损失 + 可选GAN。

### A.4.4 优化收敛条件

| 条件 | 表述 |
|-----------|-----------|
| 一阶（必要条件） | ∇L(θ*) = 0 |
| 二阶（充分条件） | ∇²L(θ*) ≻ 0（正定Hessian矩阵） |
| 凸性 | f(λx + (1-λ)y) ≤ λf(x) + (1-λ)f(y) |

满秩 **A** 的线性最小二乘具有唯一全局最小值。大多数深度学习训练问题是非凸的；在实践中收敛到局部最小值通常已足够。

矩阵求逆的数值稳定性与条件数（condition number）密切相关。在 ISP 的 CCM（色彩校正矩阵）拟合中，当用于拟合的 ColorChecker 测量矩阵 A 的条件数 κ(A) 很大时，微小的测量误差会被放大。请解释：条件数的定义是什么（σ_max / σ_min）？当 κ(A) > 100 时，对 CCM 求解结果意味着什么？在实际标定中，如何通过增加色块数量或添加正则化项来改善条件数？

傅里叶变换在 ISP 图像质量分析中有重要应用：MTF（调制传递函数）可以通过 LSF（线扩散函数）的傅里叶变换得到，图像的频域功率谱可以反映噪声特性（白噪声对应平坦频谱）。请分析：如何用 numpy.fft 计算一幅图像的 2D 功率谱密度（PSD）？在 PSD 图中，JPEG 压缩伪影（块效应）和传感器随机噪声分别呈现怎样不同的频域特征？

**练习 3（实践）**
使用 NumPy 验证 sRGB → XYZ 矩阵乘法的正确性。已知 IEC 61966-2-1 标准定义的 sRGB → XYZ_D65 变换矩阵 M（3×3），编写 Python 代码：（1）将 sRGB 白点 [1.0, 1.0, 1.0] 变换到 XYZ 空间，验证结果是否接近 D65 白点 [0.9505, 1.0000, 1.0890]；（2）将 sRGB 红色 [1.0, 0.0, 0.0] 变换到 XYZ，计算对应的 xy 色度坐标，验证是否落在 sRGB 色域的红色顶点处。
